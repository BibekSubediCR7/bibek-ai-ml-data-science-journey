# Bibek AI/ML/Data Science Journey

This repository documents my journey in learning Artificial Intelligence, Machine Learning, and Data Science. It is structured to track my progress from foundational concepts to advanced topics and real-world projects.

## Purpose

The goal of this repository is to maintain a well-organized record of everything I learn, including code, notes, and practical implementations. It serves as both a personal reference and a demonstration of my growth over time.

## Structure

The repository is organized into topic-based folders such as:

- **01-Python** – Core programming concepts and practice  
- **02-NumPy** – Numerical computing fundamentals  
- **03-Pandas** – Data manipulation and analysis  
- **04-Data-Cleaning** – Data preprocessing techniques  
- **05-Data-Visualization** – Exploring and presenting data  
- **06-Projects** – Real-world applications and analysis  

As I continue learning, more folders will be added, including Machine Learning, Deep Learning, Mathematics for AI, Statistics, and other advanced topics.

## What You Will Find

- Well-structured code for each topic  
- Jupyter notebooks with explanations and examples  
- Practice exercises and implementations  
- Project-based learning with real datasets  

## Ongoing Work

This repository is continuously updated as I learn new concepts and expand my knowledge in AI, Machine Learning, and Data Science.
